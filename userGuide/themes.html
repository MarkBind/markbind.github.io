    Themes

    MarkBind supports the ability to style your website with a variety of themes.

    Specifying a Theme

    You can specify a theme for your site by using the theme property of the site.json. For example, to apply the Cerulean theme, add "theme": "bootswatch-cerulean" to your site.json.

    If no theme property is specified, your site will be styled with default Bootstrap theme.

    Supported Themes

    Currently, MarkBind supports all light themes from Bootswatch. Visit each of the theme pages below to see how different visual components are styled.
